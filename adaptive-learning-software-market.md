<p><ul><li>Cloud Based</li><li>On-Premises</li></ul></p>
<p><p>Adaptive Learning Software is classified into two market types: Cloud Based and On-Premises. Cloud-based adaptive learning software operates on remote servers, allowing users to access and use the software via the internet. It offers the advantage of scalability, ease of implementation, and reduces the need for hardware resources. On the other hand, On-Premises adaptive learning software requires installation on local servers, allowing users to have control and customization over the software, but it requires dedicated hardware resources and maintenance.</p></p>
